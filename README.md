# HR-Analysis-Workforce-Compensation-Retention
## Overview

This project analyzes workforce distribution, employee compensation, and employee retention trends using Excel and Power BI.

## Objective

To evaluate workforce distribution, employee compensation, and retention trends to identify factors influencing organizational performance.

## Tools Used

* Microsoft Excel
* Power BI Desktop
* Power Query
* DAX

## Key Features

* Data Cleaning and Preparation
* Pivot Table Analysis
* Interactive Power BI Dashboards
* KPI Cards and Slicers
* Calculated Columns and Measures
* Bookmarks and Drill-through Navigation

## Dashboards

1. Workforce Overview
2. Compensation Analysis
3. Retention Analysis
4. Department Details

## Files Included

* Raw Dataset
* Cleaned Dataset (HR Dataset - Mini project)
* Data Cleaning & Modeling Report (Mini project 1 - report)
* Power BI (Mini Project - Dashboard)
